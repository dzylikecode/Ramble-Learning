# 数组对象

基本数据类型只有一个值，而引用数据类型可以含有多个值。

```Javascript
var arr = new Array("HTML","CSS","JavaScript","jQuery","Vue.js");

var 数组名 = new Array(元素 1, 元素 2, ……, 元素n); //完整形式 var 数组名 = [元素 1, 元素 2, ……, 元素n];          //简写形式
```
其实一个数组是可以存储“不同数据类型”的数据的，只不过我们极少那样做。一般情况下都是用数组来存储“相同数据类型”的数据

- clip

slice

- head
add : unshift

del :  shift

- tail

add : push

del : pop

- connect

join
